
# Minimalist-Theme

## How to install

To install this theme you need to know how to build omega or upsilon and add these arguments

```bash
  $ make THEME_NAME=minimalist_oiha THEME_REPO=https://github.com/Oiha-dev/Minimalist-Theme
```
